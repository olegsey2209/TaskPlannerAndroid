# Планировщик (TaskPlanner)

Приложение для планирования личных дел для iOS и Android.

## Требования

* Android Studio Hedgehog или новее
* Android SDK 35
* Android-устройство или эмулятор (Android 12.0 и новее)

## Установка и запуск

1. Клонировать данный репозиторий.
2. Открыть проект в Android Studio.
3. Дождаться завершения синхронизации Gradle.
4. Подключить Android-устройство или запустить эмулятор.
Нажать кнопку Run.

После сборки приложение будет установлено и запущено на выбранном устройстве.
